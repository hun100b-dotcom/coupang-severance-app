# 프리미엄 마이페이지(대시보드) UI 이식 및 라우팅 연결

## 1. 현황 분석

| 항목 | 내용 |
|------|------|
| **라우팅** | `App.tsx`: `/`, `/severance`, `/unemployment`만 존재. `/mypage` 없음. |
| **메인 홈** | `Intro.tsx`: 전체가 `min-h-screen flex` 중앙 정렬. 상단에 별도 헤더 없음. |
| **인증** | 전역 Auth Context 없음. `NonEligibleResult`에만 카카오/구글 목업 로그인 존재. |
| **UI** | `lucide-react` 사용 가능 (Bell, Settings, User, ChevronRight 등). |

## 2. 작업 목표

1. **신규 페이지**: `src/pages/MyPage.tsx` — V3 헤더 + V2 바디 Pixel-Perfect 반영.
2. **라우팅**: `App.tsx`에 `/mypage` 추가.
3. **홈 진입점**: `Intro.tsx` 상단 우측에 "My" + User 아이콘, 클릭 시 `/mypage` 이동.
4. **인증**: 로그인 시 마이페이지 본문, 비로그인 시 카카오/Google 로그인 두 스텝만 노출.

## 3. 디자인 명세 (Pixel-Perfect)

- **헤더**: `bg-[#F2F4F6]/90`, `backdrop-blur-xl`, `sticky top-0`, `z-30`. 중앙 "내 정보". 우측 Bell(알림)+Settings. 알림 뱃지 `w-2.5 h-2.5 bg-red-500`.
- **카드**: `bg-white`, `rounded-[32px]`, `shadow-[0_12px_40px_rgba(0,0,0,0.04)]`, `border border-gray-100/50`.
- **D-Day 바**: `bg-gradient-to-r from-blue-400 to-[#3182F6]`, 우측 glow `bg-white/30`.
- **CATCH PRO**: `bg-gradient-to-br from-[#1c1c1e] via-[#2c2c2e] to-[#1c1c1e]`, 텍스트 `#E5D88A`.

## 4. 생성·수정 파일

| 파일 | 작업 |
|------|------|
| `src/contexts/AuthContext.tsx` | 신규. isLoggedIn, userName, login(Kakao|Google) 목업, logout. |
| `src/pages/MyPage.tsx` | 신규. 헤더+프로필+D-Day+진단리포트+PRO+알림. 비로그인 시 로그인 스텝. |
| `App.tsx` | AuthProvider 래핑, Route `/mypage` 추가. |
| `Intro.tsx` | 상단 sticky 헤더 우측 "My" + User, navigate('/mypage'). |
