# 쿠팡 일용직 퇴직금 계산기 - 실행 방법

## 1. 필요한 라이브러리 설치

터미널(명령 프롬프트)을 열고, 프로젝트 폴더로 이동한 뒤 아래 명령어를 실행하세요.

```bash
cd coupang_severance_app
pip install -r requirements.txt
```

**※ `externally-managed-environment` 오류가 나는 경우** (Linux/Mac 등): 가상환경을 사용하세요.

```bash
cd coupang_severance_app
python3 -m venv .venv
# Windows:  .venv\Scripts\activate
# Mac/Linux: source .venv/bin/activate
pip install -r requirements.txt
```

또는 라이브러리를 하나씩 설치하려면:

```bash
pip install streamlit pandas openpyxl plotly
```

- **streamlit**: 웹 앱 실행
- **pandas**: 엑셀/CSV 읽기 및 데이터 처리
- **openpyxl**: 엑셀(.xlsx) 파일 읽기
- **plotly**: 그래프 시각화

---

## 2. 앱 실행

같은 폴더에서 아래 명령어를 입력합니다. (가상환경 사용 시에는 먼저 `source .venv/bin/activate` 등으로 활성화한 뒤 실행)

```bash
streamlit run app.py
```

실행이 되면 터미널에 예를 들어 다음과 같은 메시지가 나옵니다.

```
  You can now view your Streamlit app in your browser.

  Local URL: http://localhost:8501
  Network URL: http://192.168.x.x:8501
```

---

## 3. 브라우저에서 사용하기

1. 브라우저를 열고 주소창에 **http://localhost:8501** 을 입력해 접속합니다.
2. **「근로내역 파일 업로드」**에서 엑셀(.xlsx) 또는 CSV(.csv) 파일을 선택합니다.
3. 퇴직 기준일을 선택한 뒤, 화면에 표시되는 **평균임금**, **예상 퇴직금**, **최근 3개월 급여 추이** 등을 확인합니다.

---

## 4. 종료 방법

앱을 종료하려면 터미널에서 **Ctrl + C** 를 누릅니다.
