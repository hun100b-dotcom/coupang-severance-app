# 근로내역서 발급 가이드 이미지

이 폴더에 아래 이미지를 넣으면 앱의 "근로내역서는 어디서 받나요?" 가이드에 사용됩니다.

- **guide_step1.png** — 증명원 신청/발급·증명원 출력 화면
- **guide_step2.png** — 고용·산재보험 자격 이력 내역서 폼 (보험구분 고용, 조회구분 일용)
- **guide_step3.png** — 조회/신청 단계
- **guide_step4.png** — 일용근로·노무제공내역서 PDF 미리보기

`guide_stepN_hl.png` (강조 버전)이 있으면 해당 파일을 우선 사용합니다.
위 파일이 없으면 `assets/` 폴더에서 동일한 내용의 이미지 파일명으로 검색합니다.
