# 퇴직금 한번에 — 통합 플랜 보고서 (에이전트 실행용)

**문서 목적**: 플랜 모드에서 정리한 모든 내용을 한 번에 실행할 수 있도록 정리한 보고서.  
**대상**: 쿠팡/물류 일용직 퇴직금·실업급여 확인 Streamlit 앱.

---

## 1. 개요 및 최종 결정 사항

- **정밀계산 업로드**: **PDF만** 허용. 근로복지공단 일용근로내역서 PDF만 받고, 엑셀·CSV 업로드는 제거.
- **문구 톤**: 전반 **구어체**, 친밀감. 가이드 제목은 **「근로내역서는 어디서 받나요?」** 같은 질문형.
- **실업급여**: 통합 인트로에서 **퇴직금 확인하기 / 실업급여 확인하기** 두 CTA로 선택 후 각 플로우 진입. (실업급여 플로우는 준비 중 안내)
- **가이드 이미지**: 사용자 제공 4장 활용. 개인정보는 **전부 마스킹**, 클릭 필요한 부분은 **토스 느낌으로 하이라이트**.

---

## 2. UI/UX 개선

### 2.1 메인 인트로 화면 (최우선)

- **위치**: `app.py` — `init_session_state`, `main()` 분기, `render_intro()`.
- **플로우**: 앱 진입 → **인트로 1장** → CTA 선택 시 해당 플로우로 이동.
- **인트로 구성**
  - 큰 이모지 (상단 중앙, 4rem~5rem).
  - 헤드라인: "일용직도 퇴직금·실업급여 받을 수 있을까?" (구어체).
  - 서브카피: "쿠팡·컬리·CJ 등에서 일하셨다면, 자격 확인부터 계산까지 도와드려요."
  - **두 개 CTA**: [퇴직금 확인하기], [실업급여 확인하기].
- **세션**: `intro_done`, `service` (= "severance" | "unemployment") 추가.

### 2.2 친근한 톤 디자인 (웜 톤)

- **컬러**: 페이지 배경 `#f7f6f3`, 카드 `#fffefc`, 테두리 웜 그레이(`#e8e6e3`), 본문/보조 텍스트 웜 톤.
- **헤더·카드**: 웜 화이트·웜 테두리·그림자.
- **버튼**: primary 블루, secondary 웜 배경·웜 테두리.
- **Success**: 연한 그린 틴트 배경.

### 2.3 문구 톤 통일

- 앱 전반 **구어체** (~해요, ~주세요, ~있어요).

---

## 3. 실업급여 통합

- **진입**: 인트로에서 [실업급여 확인하기] 클릭 시 `service = "unemployment"` 저장 후 "준비 중" 안내 화면.
- **전환**: "퇴직금 확인하러 가기" 버튼으로 퇴직금 플로우로 이동 가능.

---

## 4. 정밀계산 — PDF 전용

### 4.1 업로드

- **허용 형식**: `type=["pdf"]` 만.
- **안내 문구**: "일용근로내역서 PDF를 올려 주세요" (구어체).

### 4.2 PDF 파싱 및 계산 반영

- **라이브러리**: `pdfplumber`.
- **함수**: `parse_welcomwel_pdf(uploaded_file)` — PDF 테이블 추출 → 근로년월, 사업장명, 근로일자, 근로일수, 임금총액 → 일별 DataFrame(근무일, 지급액, 사업장) 반환.
- **분기**: PDF 업로드 시 `parse_welcomwel_pdf` 호출 후 기존 계산 파이프라인 연결.

## 5. 가이드 — 「근로내역서는 어디서 받나요?」

### 5.1 구성

- **제목**: 「근로내역서는 어디서 받나요?」
- **배치**: 정밀계산 화면 파일 업로더 위쪽 expander.
- **단계**: Step 1 로그인 → Step 2 메뉴 선택 → Step 3 조회(고용/일용) → Step 4 신청·증명원 출력.
- **이미지**: `assets/guide/guide_step1.png` ~ `guide_step4.png`.

### 5.2 가이드 이미지 — 마스킹·하이라이트

**마스킹 (모든 이미지)**  
- 이름, 주민등록번호, 접수번호, 사업장명, 발급인, 세션 시간 등 개인정보 전부 마스킹.

**하이라이트 (토스 느낌)**  
- 색: 토스 블루(#3182f6), 둥근 사각형 또는 부드러운 glow.
- 이미지별: (1) 자주 찾는 서비스 중 "고용·산재보험 자격 이력 내역서" 카드 (2) 증명원 신청/발급 탭, 왼쪽 메뉴, 고용/일용, 조회 버튼 (3) 자격관리 상세이력 선택 체크박스, 신청 버튼 (4) 증명원 출력 버튼.
